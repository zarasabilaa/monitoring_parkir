1. Cara Penggunaan Sistem
Sistem monitoring parkir digunakan dengan cara menjalankan MongoDB dan server Flask. Dalam sistem ini Pengguna dapat melakukan scan QR Code untuk masuk ke area parkir, lalu sistem akan otomatis mencari slot kosong, menyimpan data kendaraan ke database, dan mengubah status slot menjadi terisi. Jika QR Code yang sama discan kembali, sistem akan mendeteksi kendaraan keluar, menghitung durasi parkir, serta mengosongkan kembali slot parkir. Status slot parkir dapat dilihat melalui endpoint GET /api/slots, histori parkir dapat diakses melalui GET /get_logs, dan reset sistem dapat dilakukan menggunakan endpoint POST /reset.
2. Konsep Web Service pada Sistem
Sistem monitoring parkir ini menggunakan konsep web service berbasis REST API yang berfungsi sebagai penghubung antara frontend dan database menggunakan protokol HTTP dan format data JSON. Ketika pengguna melakukan scan QR Code atau membuka dashboard monitoring, frontend akan mengirim request ke server Flask melalui endpoint API, kemudian server memproses data ke MongoDB dan mengembalikan response berupa JSON kepada frontend. Implementasi web service dilakukan menggunakan Flask route seperti endpoint /api/slots, /scan, /get_logs, dan /reset yang digunakan untuk mengambil data slot parkir, menyimpan aktivitas kendaraan, melihat histori parkir, serta mereset sistem. Dengan konsep web service, sistem menjadi lebih fleksibel, realtime, dan mudah diintegrasikan dengan aplikasi lain seperti mobile app maupun sistem IoT.
